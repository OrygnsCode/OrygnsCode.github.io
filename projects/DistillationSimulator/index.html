McCabe-Thiele Diagram

Theory

The McCabe-Thiele method is a graphical method for determining the number of theoretical plates required for a binary distillation.

Key equations:

Rectifying section operating line: \( y_{n+1} = \frac{R}{R+1}x_n + \frac{x_D}{R+1} \)

Stripping section operating line: \( y_{m+1} = \frac{L'}{V'}x_m - \frac{B x_B}{V'} \)

Feed line (q-line): \( y = \frac{q}{q-1}x - \frac{x_F}{q-1} \)
